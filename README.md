# Sales & Customer Analytics — Northwind Database

Business analysis of a retail sales database using MySQL.

Covers customer behaviour, revenue by product category, 
high-cost shipment identification, inactive customer detection, 
and multi-table reporting across orders, products, and suppliers.

Techniques: JOINs, GROUP BY, HAVING, subqueries, window functions, CTEs, DATE functions.

Tool: MySQL 8.0 | Dataset: Northwind (mywind)
